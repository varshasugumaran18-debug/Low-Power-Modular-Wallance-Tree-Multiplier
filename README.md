# Low-Power Modular Wallace Tree Multiplier

# Description
Designed and implemented a 16-bit Low-Power Modular Wallace Tree Multiplier using an optimized 7:3 counter and efficient adders in Verilog HDL. The design reduces power consumption and hardware complexity while improving multiplication speed. Verified through ModelSim simulations and FPGA implementation on the DE10-Lite board.

## Tools Used
- Verilog HDL
- ModelSim
- FPGA
